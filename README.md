# The Weather Project

      ***Description:***

The Weather Forecast and [ForecastNow.com](http://forecastnow.com/) provide a national and local weather forecast for cities, as well as weather radar, report and hurricane coverage.

***Stack:***

***Dependencies*** 

Axios - dotenv  - Api
